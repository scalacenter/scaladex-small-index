scala-js-dookie

Scala.js facades for dookie.

Generated from TypeScript definitions based on dookie 0.1.

Usage

First, add the JCenter resolver to your build file:

resolvers += Resolver.jcenterRepo,

Then, add it to your SBT dependencies:

libraryDependencies += "com.definitelyscala" %%% "scala-js-dookie" % "1.0.2"

Classes and traits are available in the package com.definitelyscala.dookie, scaladoc is provided.

This project provides Scala.js facades. You will still need to include the JavaScript library source in your web page.
